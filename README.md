AI Agent 代码重构系统
基于多智能体协同架构的自动化代码重构系统，可智能完成代码分析、重构、测试与报告全流程工作。

项目概述
本系统采用模块化多智能体协作架构，实现了完整的自动化代码重构流水线。通过 Planner-Agent 拆解复杂重构任务，调度 Analyzer-Agent 扫描代码潜在问题（冗余逻辑、未使用变量、调试残留代码等），由 Refactor-Agent 基于规则生成结构化重构建议，Tester-Agent 自动执行单元测试验证改动有效性，最终由 Reporter-Agent 汇总生成类 PR 格式的结果报告。

核心功能
多智能体协同工作流：专业分工的智能体分别处理重构不同阶段，职责清晰、可独立迭代
智能代码问题检测：自动识别冗余逻辑、未使用变量、调试代码、不规范命名等常见代码问题
规则化重构生成：基于统一编码规范生成结构化、一致性的代码优化方案
自动化测试验证：自动执行单元测试，确保重构后代码逻辑正确性
PR 风格报告输出：生成清晰易读的重构报告，包含改动说明、代码对比和测试结果
高可扩展架构：支持快速新增智能体、自定义重构规则和检测逻辑

架构设计
系统采用线性流水线式多智能体架构，各模块职责单一、解耦性强：

任务输入 → 规划智能体(Planner) → 分析智能体(Analyzer) → 重构智能体(Refactor) → 测试智能体(Tester) → 报告智能体(Reporter) → 输出重构报告

 各智能体职责
| 智能体名称 | 核心职责 |
|------------|----------|
| Planner-Agent | 任务拆解、工作流编排与各智能体调度 |
| Analyzer-Agent | 代码语法扫描、语义分析与问题检测 |
| Refactor-Agent | 基于编码规范生成结构化重构代码与建议 |
| Tester-Agent | 自动执行单元测试，验证重构后代码逻辑一致性 |
| Reporter-Agent | 汇总全流程结果，生成类 PR 格式的可视化报告 |

快速开始
 前置条件
- Python 3.8 及以上版本
- pip 包管理工具

 安装步骤
1. 克隆仓库到本地：
bash
git clone https://github.com/paddlefeng/ai_agent_refactor.git
cd ai_agent_refactor

2. 安装项目依赖：
bash
pip install -r requirements.txt

 运行示例
执行以下命令，系统将自动对 `sample_code.py` 中的示例代码进行重构，并输出完整报告：
bash
python app.py

使用指南
 基础使用
1. 将 `sample_code.py` 中的内容替换为你需要重构的代码
2. 更新 `test_sample.py` 为对应代码的单元测试用例（可选但推荐）
3. 运行重构流水线：
bash
python app.py

 自定义配置
修改重构规则：编辑 `agents/refactor.py` 中的提示词模板，调整重构标准和风格
新增代码检测项：扩展 `agents/analyzer.py` 中的分析逻辑，添加自定义问题检测规则
自定义报告格式：修改 `agents/reporter.py` 中的报告生成逻辑，适配团队文档规范

扩展能力（生产环境可选）
为满足企业级生产使用需求，可扩展以下智能体模块：
Git 智能体：自动化 Git 操作（差异对比、代码提交、PR 创建）
配置智能体：集中管理团队编码规范、架构约束和重构规则
日志智能体：统计 Token 消耗、运行耗时、重构成功率等量化指标
回滚智能体：提供文件备份和自动回滚能力，保障重构过程安全
